# Parcial1-LB3-TiendaOnline
Tienda Online con ASP.NET para el Parcial 1 de Laboratorio de Programacion 3

**Alumno:** Segui Marcos
**Materia:** Laboratorio de Programación 3

## Instrucciones para ejecutar el proyecto:
1. Descargar el repositorio.
2. Abrir SQL Server y ejecutar el archivo `script_db.sql` incluido en la raíz. Esto creará la base de datos `TiendaOnline` con sus tablas y datos de prueba.
3. Abrir la solución `.sln` en Visual Studio.
4. Verificar en el `Web.config` que la cadena de conexión coincida con su instancia local (actualmente configurada para `(localdb)\MSSQLLocalDB`).
5. Ejecutar la página `TiendaOnline.aspx`.
